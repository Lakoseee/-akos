Program to prosta gra snake w c++. Celem gracza jest zdobyćie jak najwiekszej liczby punktów zbierając owoce. Wejście w ściane lub we własny ogon kończy gre.
Wężem steruje sie przy pomocy "wasd" a klawiszem "q" można zakończyć działanie programu.
Kompilacja i uruchomienie programu:
  make - kompiluje pliki
  make run - odpala gre snake
  make clean - usuwa wszystie pliki binarne
